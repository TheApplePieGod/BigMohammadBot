```ini
[$report]
- Calculates and displays a report of user activity for the previous week

[$reportnow]
- Calculates and displays a report of user activity for the current week

[$reportall]
- Calculates and displays a report of user activity for all time

[$stats]
- Displays most statistics about activity data from last week

[$statsnow]
- Displays most statistics about activity data from this week

[$statsall]
- Displays most statistics about activity data from all time

[$reportchart (num weeks) (@user1) (@user2) ...]
- Displays a visualization of the mentioned users' activity history
- 1-5 users may be requested at a time
- Specify 0 for 'num weeks' for an all time chart
- Please do not spam this command
```
